# <a name="ef-core-tools--extensions"></a>EF podstawowe narzędzia & rozszerzenia

Narzędzia i rozszerzenia oferowanie dodatkowych funkcji Entity Framework podstawowych.

Rozszerzenia są tworzone przez różnych źródeł. Nie wszystkie rozszerzenia są obsługiwane w ramach projektu Entity Framework Core. Rozważając rozszerzenia innych firm, należy ocenić jakości, licencjonowanie, obsługi, itp., aby upewnić się, że spełniają one wymagania.
